# 🏦 Bank Account Management System

## 📌 Overview
This is a simple Bank Account Management System built using Python. It allows users to create an account, log in, and perform basic banking operations through a command-line interface.

---

## 🚀 Features
- Create a new account with username and password
- Secure login system
- Deposit money
- Withdraw money
- Transfer money between accounts
- Check account balance
- Data persistence using JSON file

---

## 🛠️ Technologies Used
- Python 3
- JSON (for data storage)
